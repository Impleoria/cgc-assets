# CGC Assets — Café Grillé Collection

Dépôt dédié aux assets visuels du projet **Café Grillé Collection (CGC)**.
Géré par Impleoria pour la marque CGC de Christian Figuin.

## Structure

```
images/
├── produits/   ← designs originaux (27 produits)
├── posts/      ← visuels traités 1080×1080 pour publication
└── logos/      ← logos et assets marque
```

## URL Pattern

```
https://raw.githubusercontent.com/Impleoria/cgc-assets/main/images/produits/nom.png
https://raw.githubusercontent.com/Impleoria/cgc-assets/main/images/posts/nom_post.png
```

## Pipeline

PC Obsidian → Obsidian Git → ce repo → Claude bash_tool → PIL 1080×1080 → Postiz → Telegram/Facebook

## Marque CGC

- Couleur brand : #b92353
- Format posts : 1080x1080px
- Plateformes : Telegram + Facebook
